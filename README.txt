In order to run this program you need to:

1. Open the project file, move to "Server side" directory, open the directory, 
open the command line from this location
and run the command "node app.js"

2. If you running this project via Chrome: First close all chrome tabs in your computer, 
open the location of the file "chrome.exe", 
open the command line from this location
and run the command "chrome.exe --allow-file-access-from-files"

3. In the projet file, move to "Client side" directory, 
open the directory and open the file "index.html" in your browser


notes: 
1. Number of views and reviews of a POI updated only after refrashing the page.
2. Adding or removing favorite point from list of favorite points happens only at POI page.
